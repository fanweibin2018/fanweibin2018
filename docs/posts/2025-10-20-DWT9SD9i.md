# **使用adb命令行配对**

1. 在手机的 设置 - 开发者选项 中启用"无线调试"
2. 打开电脑的终端，进入adb目录： `<HBuilderX安装目录>/plugins/launcher-tools/tools/adbs`。如使用低于4.61版本的HBuilderX，需自定义高版本adb，选择自己的adb路径。
3. 手机端的无线调试中，选择使用 配对码配对设备，找到您的 IP 地址、端口号和配对码。记下设备上显示的 IP 地址、端口号和配对码。
4. 在电脑的终端上，运行 `adb pair ipaddr:port`。请使用上述 IP

![](/images/external/b03baadb48-android-wifi-settings-6.png)

5. 收到提示时，输入配对码，如果命令输出`Successfully paired`, 就代表配对成功。

![](/images/external/8dfe1ffaf0-android-wifi-settings-5.png)

6. 配对设备后，你可以在HBuilderX的运行设备列表看到此设备，接下来的使用和有线adb相同。

配对成功后，在一定有效期内，无需再重复配对。在手机端打开无线调试后，过一会手机的通知栏会弹出`已连接无线调试`。然后在HBuilderX的运行界面会自动检测到该设备。

# **win10系统电脑如何将时间显示到秒**

**一、显示到秒**

1. 【Win+R】组合键打开运行窗口，输入【regedit】回车

2. 在注册表中定位到

【HKEY\_CURRENT\_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced】，在空白处右击选择【新建】【DWORD（32位）值】

3. 将新建值的名称改为ShowSecondsInSystemClock，并双击编辑数值数据，将其改为1后确定，然后关闭注册表，重启即可
