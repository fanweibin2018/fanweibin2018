# 基金相关概念

## 一、基金的基本分类

### 按投资标的分类

| 类型 | 投资什么 | 风险等级 | 举例 |
| --- | --- | --- | --- |
| **货币基金** | 短期债券、存款等 | 极低 | 余额宝 |
| **债券基金** | 国债、企业债等 | 低 | 纯债基金 |
| **混合基金** | 股票+债券混合 | 中 | 偏股混合、偏债混合 |
| **股票基金** | 80%以上投股票 | 高 | 沪深300指数基金 |
| **商品基金** | 黄金、白银、原油等 | 高 | 黄金ETF、白银期货基金 |
| **QDII基金** | 境外资产 | 中高 | 纳斯达克指数基金 |

---

## 二、交易场所相关术语

### 场内 vs 场外

| 概念 | 含义 | 交易方式 | 开户要求 |
| --- | --- | --- | --- |
| **场内** | 在证券交易所交易 | 像股票一样买卖，实时成交 | 需要证券账户（股票账户） |
| **场外** | 不在交易所，通过银行/支付宝/天天基金等 | 按当日净值申购赎回，T+1确认 | 不需要证券账户 |

---

## 三、ETF / LOF / 联接基金

### ETF（交易型开放式指数基金）

```
ETF = Exchange Traded Fund = 交易所交易基金
```

| 特点 | 说明 |
| --- | --- |
| 交易方式 | **只能场内**，像股票一样实时买卖 |
| 价格 | 实时变动，有买一卖一价 |
| 费用 | 只收券商佣金（万1~万3），无申购赎回费 |
| 最小单位 | 100份起（约几百元） |
| 典型产品 | 华夏黄金ETF（518850）、沪深300ETF |

**优点**：费率低、交易灵活、透明度高 **缺点**：必须有证券账户

---

### LOF（上市开放式基金）

```
LOF = Listed Open-ended Fund = 上市开放式基金
```

| 特点 | 说明 |
| --- | --- |
| 交易方式 | **既能场内也能场外**，两边都可以 |
| 场内 | 像股票一样买卖 |
| 场外 | 像普通基金一样申购赎回 |
| 套利 | 场内场外价格可能不同，存在套利空间 |
| 典型产品 | 国投瑞银白银期货LOF（161226） |

**LOF的风险**：场内价格可能远高于净值（溢价），买贵了会亏钱

---

### 联接基金

```
联接基金 = 场外版的ETF
```

| 特点 | 说明 |
| --- | --- |
| 本质 | 90%以上资产投资于对应的ETF |
| 交易方式 | **只能场外**（支付宝、银行、天天基金等） |
| 作用 | 让没有证券账户的人也能间接买ETF |
| 费用 | 比ETF高（多一层管理费+销售服务费） |
| 典型产品 | 华安黄金ETF联接C（支付宝里的"华安黄金"） |

**关系图**：

```
你 → 购买联接基金 → 联接基金购买ETF → ETF持有黄金/股票等
```

## 四、A类 / C类基金

同一只基金分成不同份额类别，**区别在于收费方式**：

| 份额 | 申购费 | 赎回费 | 销售服务费 | 适合人群 |
| --- | --- | --- | --- | --- |
| **A类** | 有（约0.1%-1.5%） | 有（持有越久越低） | 无 | **长期持有**（1年以上） |
| **C类** | 无 | 有（7天内1.5%，30天后常免） | 有（约0.2%-0.4%/年） | **短期持有**（几天到几个月） |

### 选择口诀

- **持有超过1-2年** → 选A类（一次性付申购费，没有每年的销售服务费）
- **持有不到1年** → 选C类（免申购费，虽然有年费但总成本更低）

---

## 五、期货基金 vs 现货基金

| 类型 | 投资标的 | 特点 | 举例 |
| --- | --- | --- | --- |
| **现货基金** | 直接持有实物或现货合约 | 跟踪价格更紧密，费用低 | 黄金ETF（持有上海金交所黄金） |
| **期货基金** | 持有期货合约 | 有展期成本，跟踪误差大 | 国投瑞银白银期货LOF |

**期货基金的问题**：

- 期货合约有到期日，需要不断"换月"（展期）
- 展期会产生成本，长期持有可能跑输现货价格

---

## 六、指数基金 vs 主动基金

| 类型 | 管理方式 | 费率 | 目标 |
| --- | --- | --- | --- |
| **指数基金（被动）** | 复制指数成分股 | 低（0.5%左右） | 跟上指数，不求超越 |
| **主动基金** | 基金经理选股 | 高（1.5%左右） | 试图跑赢市场 |

---

## 七、股票相关术语

| 术语 | 含义 |
| --- | --- |
| **A股** | 中国大陆上市、人民币交易的股票 |
| **港股** | 香港上市的股票 |
| **美股** | 美国上市的股票 |
| **T+1** | 今天买，明天才能卖（A股规则） |
| **T+0** | 今天买，今天就能卖（港股、黄金ETF等） |
| **涨跌停** | A股单日涨跌幅限制10%（科创板/创业板20%） |
| **市值** | 股价 × 总股数 = 公司值多少钱 |
| **PE（市盈率）** | 股价 ÷ 每股收益 = 多少年回本 |

## 八、一张图总结基金类型

```
                    ┌─ 货币基金（余额宝）
                    ├─ 债券基金
        按投资标的 ─┼─ 混合基金
                    ├─ 股票基金
                    └─ 商品基金（黄金、白银等）

                    ┌─ 场内（需证券账户）─┬─ ETF
                    │                      └─ LOF（也可场内）
        按交易场所 ─┤
                    │                      ┌─ 普通开放式基金
                    └─ 场外（支付宝等）───┼─ 联接基金
                                           └─ LOF（也可场外）

                    ┌─ A类（适合长期）
        按收费方式 ─┼─ C类（适合短期）
                    └─ 其他（B类、E类等较少见）
```
