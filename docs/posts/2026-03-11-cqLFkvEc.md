# Ubuntu 24.04 服务器初始化配置指南

> 适用场景：VMware 虚拟机 / 云服务器初始化，快速完成静态 IP、root 账户、防火墙、镜像源四项基础配置。

---

## 环境信息

| 项目 | 值 |
| --- | --- |
| 系统 | Ubuntu 24.04 LTS |
| 网卡 | ens33 |
| 固定 IP | 192.168.199.136/24 |
| 网关 | 192.168.199.2 |

---

## 一、配置固定 IP

Ubuntu 24.04 使用 **Netplan** 管理网络，配置文件位于 `/etc/netplan/`。

### 查看网卡名称

```
ip a
```

### 查看现有配置文件

```
ls /etc/netplan/
# 输出示例：50-cloud-init.yaml
```

### 写入静态 IP 配置

使用 `tee` 命令直接覆盖，无需编辑器：

```
sudo tee /etc/netplan/50-cloud-init.yaml > /dev/null << 'EOF'
network:
  version: 2
  renderer: networkd
  ethernets:
    ens33:
      dhcp4: no
      addresses:
        - 192.168.199.136/24
      routes:
        - to: default
          via: 192.168.199.2
      nameservers:
        addresses: [114.114.114.114, 8.8.8.8]
EOF

sudo chmod 600 /etc/netplan/50-cloud-init.yaml
sudo netplan apply
```

### 验证

```
ip a show ens33
```

输出中出现以下内容即为成功：

- `inet 192.168.199.136/24`
- `valid_lft forever`（永久有效，无 `dynamic` 字样）

## 二、启用 root 账户

### 设置 root 密码

```
sudo passwd root
```

### 允许 root SSH 登录

创建独立配置片段，不修改主配置文件：

```
sudo tee /etc/ssh/sshd_config.d/root.conf > /dev/null << 'EOF'
PermitRootLogin yes
PasswordAuthentication yes
EOF

sudo systemctl restart ssh
```

切换到 root：

```
su - root
# 或
sudo -i
```

---

## 三、关闭防火墙

Ubuntu 24.04 最小化安装默认**不包含 ufw**，无需操作。

验证方式：

```
ufw status
# 若提示 command not found，说明防火墙未安装，所有端口默认开放
```

若系统已安装 ufw，执行以下命令永久关闭：

```
sudo ufw disable
sudo systemctl disable ufw
```

## 四、配置阿里云 Ubuntu 镜像源

Ubuntu 24.04（Noble）的 APT 源配置文件格式有所变化，使用新格式 `/etc/apt/sources.list.d/ubuntu.sources`。

### 备份原始源

```
sudo cp /etc/apt/sources.list.d/ubuntu.sources /etc/apt/sources.list.d/ubuntu.sources.bak
```

### 替换为阿里云镜像

```
sudo tee /etc/apt/sources.list.d/ubuntu.sources > /dev/null << 'EOF'
Types: deb
URIs: https://mirrors.aliyun.com/ubuntu
Suites: noble noble-updates noble-backports
Components: main restricted universe multiverse
Signed-By: /usr/share/keyrings/ubuntu-archive-keyring.gpg

Types: deb
URIs: https://mirrors.aliyun.com/ubuntu
Suites: noble-security
Components: main restricted universe multiverse
Signed-By: /usr/share/keyrings/ubuntu-archive-keyring.gpg
EOF
```

### 更新软件包索引

```
sudo apt update
```

### 验证镜像源

```
apt-cache policy | grep mirrors.aliyun.com | head -5
```

输出中出现 `mirrors.aliyun.com` 即配置成功。

---

## 五、重启并全面验证

```
sudo reboot
```

重启后逐项检查：

```
# 1. 静态 IP
ip a show ens33

# 2. root SSH 登录（在另一终端测试）
ssh root@192.168.199.136

# 3. 防火墙状态
ufw status

# 4. 镜像源可用性
sudo apt update
```
